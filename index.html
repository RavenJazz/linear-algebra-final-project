Four Fundamental Spaces Finder

A calculator to help you solve for the four fundamental spaces in Linear Algebra.

Column Space

This space consists of all possible linear combinations of the columns of A. It represents the set of all vectors b for which the system Ax = b has a solution.

Null Space

The null space contains all vectors x that satisfy the equation Ax = 0. It describes the relationships and dependencies between the columns of the matrix.

Row Space

This is the span of all the row vectors of A, or the column space of A transposed. It occupies the same dimension as the column space, representing the "rank" of the matrix.

Left Null Space

This space consists of all vectors y such that A^Ty = 0 (or y^TA = 0). It is the orthogonal complement to the column space and helps describe the constraints on the output space.
